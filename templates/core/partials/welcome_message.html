Welcome. This web application reproduces much of the functionality of Epi Info 7's (for Windows) Dashboard Analysis tool. Its files and documentation may be viewed in its GitHub repository.


Its statistical analyses are executed by the epiinfo Python Package.


See here for a tutorial and other examples.


Sample datasets, including the fictional Salmonellosis dataset used in the tutorial, may be downloaded from the sample_data directory in the repository.


This is a beta product that is still under evaluation. Please compare any analysis results to those of other statistical analysis tools, such as Epi Info 7.


Epi Info 7 may still be downloaded and extracted from its code repository.
